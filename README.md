# Rust TODO CLI App

A simple command-line TODO list application written in Rust.

This project allows you to:
- Add TODO items
- List all TODO items
- Mark TODOs as done
- Delete TODOs

## Features

✅ Add TODO items  
✅ List all TODOs with their status  
✅ Mark TODO as done  
✅ Delete TODOs by ID

## Usage

### Run the app

Make sure you have [Rust](https://www.rust-lang.org/tools/install) installed.
